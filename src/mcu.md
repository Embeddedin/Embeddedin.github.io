# MCU
- CPU에 디지털 신호 (On/Off),PWM, 통신과 같이 임베디드 시스템에서 많이 사용되는 주변장치을 연결하여 하나의 칩으로 제작된 형태
- CPU와 주변장치는 데이터를 주고받는데 이러한 길을 버스라고 부름

## Pheripheral

- JTAG : 개발자가 MCU 내부를 들여다보는 기능
- NVIC :인터럽트를 관리
- CLOCK : CPU가 움직이는 속도 결정
- ADC : 아날로그 전압 → 디지털 값
- I2C
- GPIO : General Purpose Input Output
- RAM 
- T/C
- UART
